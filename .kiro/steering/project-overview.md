# Context Flow — Project Overview

## Product Identity

- **Name:** Context Flow
- **Type:** AI-powered VS Code Extension
- **Development IDE:** Kiro IDE
- **Language:** TypeScript (strict mode)
- **Runtime:** Node.js
- **Extension Platform:** VS Code Extension API

## Product Vision

Context Flow is an intelligent AI development layer that improves AI-assisted coding workflows by:

1. Understanding repository context automatically
2. Enhancing prompts before sending to AI providers
3. Enforcing architecture standards
4. Validating AI-generated code
5. Improving developer productivity
6. Reducing AI-generated technical debt

## Product Positioning

Context Flow is NOT another AI chatbot.
It is an **AI Development Intelligence Platform** — the intelligence layer between developers and AI coding tools.

## Core Differentiator

**Context Intelligence + AI Governance** instead of generic AI code generation.

## Target Audience

- React & React Native developers
- AI-first developers and freelancers
- Startup engineering teams
- Frontend modernization teams
- Enterprise development teams requiring AI governance
